### Pre-requisites
* go
  ```sh
  sudo snap install go --classic
  ```
* mongoDB

### Installation
1. Clone the repository
   ```sh
   git clone https://github.com/victorvcruz/social_network_project.git
   ```
2. Install go packages
   ```sh
   go mod download
   ```
